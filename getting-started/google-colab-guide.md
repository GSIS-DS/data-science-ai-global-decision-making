# Google Colab Guide

Google Colab is the default coding environment for this course.

## Open Course Notebooks

1. Go to https://colab.research.google.com.
2. Open the GitHub tab.
3. Search for GSIS-DS/data-science-ai-global-decision-making.
4. Choose a notebook from the main branch.

Direct Week 1 link:
https://colab.research.google.com/github/GSIS-DS/data-science-ai-global-decision-making/blob/main/notebooks/week-01/01_data_ai_evidence_global_decision_making.ipynb

## Save Workflow

Use File -> Save a copy in Drive, then export or sync your finished work to GitHub/LMS submission channels as instructed.
