# Google Colab Guide

Google Colab is the default coding environment for this course.

## What is Colab?

Colab is a free Jupyter notebook environment that runs in your browser. No installation required.

## Getting Started

1. Go to [colab.research.google.com](https://colab.research.google.com)
2. Sign in with your Google account
3. Click "GitHub" tab
4. Enter: `GSIS-DS-Fall-2026/data-science-ai-global-decision-making`

## Opening Notebooks

To open a specific notebook:
- Use the direct link from the README
- Or navigate through the GitHub tab in Colab

## Saving Your Work

- Go to File -> Save a copy in Drive
- Save frequently during your work
- The copy in your Drive is your working version

## Running Code

- Click the play button to run a cell
- Or press Shift+Enter
- Install packages with `!pip install [package-name]`

## Tips

- Use GPU/TPU under Runtime -> Change runtime type (not required for this course)
- Clear output before saving: Runtime -> Restart and run all (tests reproducibility)
- Download notebooks: File -> Download -> Download .ipynb
