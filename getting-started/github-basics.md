# GitHub Basics

This guide covers account creation and essential GitHub skills for the course.

## Create a GitHub Account (Step-by-Step)

1. Go to [github.com](https://github.com).
2. Click **Sign up**.
3. Enter an email address you check regularly.
4. Create a strong password.
5. Choose a username you can use professionally in class projects.
6. Verify your email address from the confirmation message GitHub sends.
7. Sign in and complete the basic profile setup.

Recommended profile settings for class:

- Add your real name (or preferred name) to your profile.
- Add a simple profile photo so teammates can identify you.
- Enable two-factor authentication (2FA) for account security.

## What is GitHub?

GitHub is a platform for version control and collaboration. You'll use it to:
- Access course materials
- Save your work
- Submit assignments
- Share code with teammates

## Essential Skills

### 1. Create a Repository
- Click the "+" in the top right -> "New repository"
- Name it descriptively
- Choose "Public" or "Private" as appropriate

### 2. Commit Changes
- Make edits to files
- Write a clear commit message
- Click "Commit changes"

### 3. Clone a Repository
- Click "Code" -> "Open in GitHub Desktop" or copy the URL
- Or use command line: `git clone [repository-url]`

### 4. Pull and Push
- Pull: Get latest changes from others
- Push: Share your changes with others
