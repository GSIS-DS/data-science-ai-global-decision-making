# Slack Setup

Slack is used as a supplementary communication channel for discussion and quick questions.

Official grades and formal announcements remain on [Sogang LMS](https://eclass.sogang.ac.kr/).

## Create a Slack Account (Step-by-Step)

1. Go to [slack.com/get-started](https://slack.com/get-started).
2. Click **Sign up with Email** (or continue with Google/Apple if preferred).
3. Enter your verification code from Slack email.
4. Set your display name and password.
5. Add a profile photo so classmates can identify you.

## Install the Slack App (Recommended)

- Desktop app: [slack.com/downloads](https://slack.com/downloads/)
- Mobile app: [slack.com/downloads/in-app](https://slack.com/downloads/in-app)

You can also use Slack in a browser.

## Join the Course Workspace

- Course workspace invite link: **To be announced**

## Suggested Settings for Class

- Turn on notifications for course channels.
- Set your full name or recognizable profile name.
- Keep direct messages and channel posts professional.
