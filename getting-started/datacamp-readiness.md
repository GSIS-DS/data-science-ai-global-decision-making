# DataCamp Readiness

If you have no programming background, DataCamp offers excellent introductory modules.

## Recommended Modules

- Introduction to Python
- Intermediate Python for Data Science
- Introduction to Pandas

## Alternative Resources

- [Python.org Tutorial](https://docs.python.org/3/tutorial/)
- [W3Schools Python](https://www.w3schools.com/python/)
- [Kaggle Learn Python](https://www.kaggle.com/learn/python)

## Self-Assessment

You're ready if you can:
- Write a variable and print it: `x = 5; print(x)`
- Use a list: `my_list = [1, 2, 3]`
- Understand a for loop (optional - we'll focus on pandas instead)

## Note

This course teaches Python as a tool for data analysis. We prioritize practical skills over programming theory. You can succeed without completing DataCamp if you're comfortable learning by doing.
