# Assignment Template

This folder provides a reusable assignment starter structure.

- starter.ipynb: reusable notebook scaffold
- AI_USE.md: AI disclosure template for material AI use

Before release, replace every bracketed prompt and state:

- learning objectives and prerequisite skills;
- estimated time and expected outputs;
- data source, license, and permitted redistribution;
- submission path, filename, and deadline;
- permitted collaboration and AI use;
- evaluation criteria and completion checks;
- technical contingency instructions.
