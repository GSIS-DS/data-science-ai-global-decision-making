# Prompt and Verification Guide

## Writing Good Prompts

1. **Be specific**: Instead of "analyze data," ask "calculate average GDP growth by region and explain what you find."

2. **Define format**: Request output in specific format (table, bullet points, code).

3. **Ask for verification**: Include "show me how to check this" in your prompt.

## Verification Steps

### For Code
- Run the code yourself
- Check output makes sense
- Verify against known values

### For Claims
- Cross-check with original sources
- Look for statistical significance
- Consider alternative explanations

### For Data
- Verify source is credible
- Check date range and coverage
- Look for documentation

### For Visualizations
- Check scales are honest
- Verify data matches claim
- Ensure all elements labeled
