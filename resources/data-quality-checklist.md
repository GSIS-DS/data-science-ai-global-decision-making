# Data Quality Checklist

Before analyzing any dataset, assess:

## Completeness

- [ ] Missing values identified and handled
- [ ] All expected observations present
- [ ] Key variables not systematically missing

## Accuracy

- [ ] Values within reasonable ranges
- [ ] Units consistent and documented
- [ ] No obvious errors or outliers

## Consistency

- [ ] Column names clear and consistent
- [ ] Categorical values standardized
- [ ] Temporal data aligned

## Documentation

- [ ] Source clearly identified
- [ ] Collection method documented
- [ ] Definitions provided
- [ ] Limitations stated

## Bias

- [ ] Selection bias considered
- [ ] Reporting bias considered
- [ ] Geographic/temporal coverage appropriate
