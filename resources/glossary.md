# Glossary

## Data Science Terms

**Observation** - A single row of data (one record)

**Variable** - A column in your dataset

**Missing value** - Data that was not recorded (NaN, NA, null)

**Outlier** - A value far from the typical range

**Correlation** - Statistical association between variables (does not imply causation)

**Regression** - Method to model relationships between variables

## AI Terms

**Large Language Model (LLM)** - AI system trained on text, can generate and explain code

**Prompt** - Input to an AI system

**Verfication** - Checking AI output against evidence or by independent means

**Bounded agent** - Structured workflow with defined inputs, tools, and human oversight
