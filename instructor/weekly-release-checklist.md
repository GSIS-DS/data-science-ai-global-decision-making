# Weekly Material Release Checklist

Use this checklist before publishing each teaching week.

- [ ] Topic, learning objective, activity, and assessed output align with the syllabus.
- [ ] Student instructions include prerequisites, estimated time, expected outputs, and completion criteria.
- [ ] Colab links use the `GSIS-DS` repository and `main` branch.
- [ ] Notebook succeeds with **Restart and run all** in a clean Colab runtime.
- [ ] Data sources, licenses, units, and synthetic-data status are explicit.
- [ ] Charts meet accessibility and source-note expectations.
- [ ] AI permissions and disclosure requirements are stated for the activity.
- [ ] Public files contain no solutions, student information, credentials, or grading keys.
- [ ] LMS dates, submission route, and contingency instructions match the repository.
- [ ] `python scripts/verify_course.py` passes.
