# Learning Outcomes

By the end of the course, students should be able to:

## Analytical Skills

1. **Formulate questions**: Create a focused, answerable question about a global business, policy, or international-affairs issue.

2. **Evaluate data sources**: Locate, inspect, document, and evaluate real-world data sources for quality, coverage, and bias.

3. **Work with data**: Use Python and pandas to import, clean, transform, merge, and summarize tabular data.

4. **Create visualizations**: Design accurate, interpretable, and ethically responsible visualizations.

5. **Apply statistical reasoning**: Use basic statistical reasoning and introductory predictive analysis appropriately.

6. **Distinguish claim types**: Recognize the difference between descriptive evidence, association, prediction, and causal claims.

7. **Conduct text analysis**: Perform basic text analysis using structured, transparent workflows.

## AI Literacy

8. **Use LLMs responsibly**: Apply LLMs productively while checking code, claims, sources, calculations, and interpretations.

9. **Design workflows**: Create and evaluate a bounded agent-like workflow that uses clearly specified inputs, tools, and human oversight.

## Communication

10. **Communicate findings**: Produce a reproducible analytical project and communicate conclusions clearly to a non-technical decision-maker.
