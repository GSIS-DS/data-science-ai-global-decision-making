# Course Schedule

| Week | Theme | Analytical Question | Key Tools/Skills | Activity | Output |
|------|-------|---------------------|------------------|----------|--------|
| 1 | Data, AI, and Evidence for Global Decision-Making | What does data tell us about global economic trends? | Course workflow, pandas basics, matplotlib | Load data, create first chart, identify observation/pattern/interpretation | Completed notebook with observations |
| 2 | Working with Data in Python | How do we access and explore international data? | pandas, CSV loading, filtering, summary stats | Practice with real-world datasets | Clean notebook with basic operations |
| 3 | Cleaning and Transforming Data | How do we handle messy real-world data? | Missing values, duplicates, type conversion, groupby | Data quality assessment exercise | Documented data transformations |
| 4 | Visualization and Analytical Storytelling | How do we communicate data insights effectively? | matplotlib, seaborn, chart design, ethical considerations | Create and critique visualizations | Well-designed charts with commentary |
| 5 | Exploring Patterns, Uncertainty, and Comparisons | How do we compare countries and identify trends? | Correlation, grouping, uncertainty visualization | Regional comparison analysis | Pattern analysis report |
| 6 | Finding and Documenting Global Data | Where do we find reliable international data? | World Bank API, data catalogs, documentation | Source evaluation exercise | Data source documentation |
| 7 | Statistical Reasoning for Analysts | How do we interpret statistical relationships? | Basic inference, confidence, significance | Statistical interpretation exercises | Statistical claims with caveats |
| 8 | Individual Practical Data Assessment and Feedback | Diagnostic assessment (not graded) | All tools from weeks 1-7 | Individual demonstration of skills | Feedback for improvement |
| 9 | Prediction for Decision-Making | How can we forecast global trends? | scikit-learn, train/test splits, accuracy | Build simple predictive model | Prediction analysis |
| 10 | Interpretable Machine Learning and Responsible Use | How do we explain predictions to stakeholders? | Model interpretation, feature importance | Interpret model outputs | Responsible prediction report |
| 11 | Text as Data: News, Reviews, Reports, and Policy Documents | How do we analyze textual policy documents? | Text processing, sentiment, topic analysis | Analyze policy document corpus | Text analysis findings |
| 12 | LLMs as Analytical Assistants | How do we use AI tools effectively? | Prompting, verification, workflow design | LLM-assisted analysis | AI workflow card |
| 13 | Bounded Agents and Human Oversight | How do we structure reliable AI workflows? | Workflow templates, verification steps | Design bounded agent workflow | Workflow documentation |
| 14 | Communicating Evidence to Decision-Makers | How do we present technical work clearly? | Executive summaries, dashboards, memo writing | Draft decision memo | Clear communication artifacts |
| 15 | Capstone Workshop and Peer Review | How do we improve our projects? | Peer review, revision | Collaborative refinement | Revised capstone materials |
| 16 | Capstone Presentations: Global Data Challenge | Showcase completed work | All tools | Present to peers and instructors | Final presentation |

## Assessment Note

Week 8 is a diagnostic assessment used to identify student strengths and gaps. It is **not graded** but provides valuable feedback before the capstone stage.
