# Course Description

This course introduces practical data-science and responsible-AI workflows for analyzing global economic, business, policy, and international-affairs questions. Students use Python and real-world datasets to clean, visualize, interpret, and communicate evidence; apply introductory predictive and text-analysis methods; and evaluate the strengths and limitations of large language models and bounded agent workflows.

Programming is a tool for answering real analytical questions, not an end in itself. Students will learn just enough Python and pandas to work with real-world data on international trade, economic development, policy outcomes, and global trends.

## Course Approach

This course emphasizes:
- Working with real-world, imperfect data sources
- Ethical and transparent analytical practices
- Clear communication of quantitative findings to non-technical audiences
- Responsible use of AI tools as assistants, not substitutes for thinking
- Reproducible research practices

## Topics Covered

- Python fundamentals for data analysis (pandas, matplotlib, seaborn)
- Data cleaning, transformation, and documentation
- Visualization principles and analytical storytelling
- Statistical reasoning and introductory predictive methods
- Text analysis for policy and business documents
- Large language models as analytical assistants
- Bounded agent workflows with human oversight

## Prerequisites

- Introductory statistics or equivalent quantitative literacy (recommended)
- No prior programming experience required

## Target Audience

Second- and third-year undergraduate students at Sogang University's Geppert School of International Studies, including those studying International Commerce, International Relations, global business, public policy, development, finance, sustainability, and international affairs.
