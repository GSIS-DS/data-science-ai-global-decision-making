# Assessment Framework

**Note: The exact weightings below may be adjusted before the final syllabus is published.**

## Assessment Components

| Component | Weight | Description |
|-----------|--------|-------------|
| Data Readiness Tasks | Completion requirement | Introductory setup and verification tasks |
| Weekly/Biweekly Labs | 20% | Hands-on exercises reinforcing key skills |
| Applied Assignment 1: Data Cleaning and Documentation | 15% | Clean and document a messy dataset |
| Applied Assignment 2: Visualization and Interpretation | 15% | Create and explain visualizations for a global issue |
| LLM Verification and Responsible AI Exercise | 10% | Demonstrate responsible AI use with verification |
| Capstone Team Project | 30% | Team project answering a real global decision question |
| Participation, Peer Review, Professional Contribution | 10% | Engagement in discussions and peer review |
| Week 8 Practical Data Assessment | 0% (diagnostic) | Individual skills diagnostic with feedback |

## Assessment Philosophy

- **Formative emphasis**: Most assignments focus on learning, not just grading
- **Collaboration**: Labs encourage teamwork; capstone is explicitly collaborative
- **Transparency**: All analytical work should be reproducible and well-documented
- **Responsibility**: AI use must be declared and verified

## Submission Guidelines

All assignments are submitted through individual private repositories generated by the assignment system. See each assignment's README for specific submission instructions.
