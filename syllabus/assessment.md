# Assessment Framework

| Component                                                        | Weight |
| ---------------------------------------------------------------- | -----: |
| Data Readiness Tasks and weekly lab engagement                   |    15% |
| Mini-project 1: Data Cleaning and Documentation                  |    10% |
| Midterm: Individual Applied Data Analysis                        |    20% |
| Mini-project 2: Visualization, Interpretation, and Communication |    10% |
| AI Verification and Text-Data Exercise                           |    15% |
| Final Team Capstone Project                                      |    25% |
| Individual Capstone Defense or Reflection                        |     5% |
| Total                                                            |   100% |

Week 8 is a graded midterm and includes developmental feedback.

The exact operational details may be adjusted before the final Spring 2027 syllabus is published, but the course design preserves the balance among individual competence, cumulative applied work, responsible AI use, and team-based capstone performance.
