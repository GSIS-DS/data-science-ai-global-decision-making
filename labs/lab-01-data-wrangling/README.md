# Lab 01: Data Wrangling

Practice inspection, transformation, filtering, and documentation without changing the raw teaching file.

- Estimated time: 60 minutes
- Prerequisite: Weeks 2-3 or equivalent pandas basics
- Output: completed notebook and a short transformation log

## Tasks

1. Inspect columns, types, missing values, and duplicate rows.
2. Create a deliberately messy copy and make justified corrections.
3. Derive trade balance and verify the calculation for one row.
4. Filter one region and summarize results by year.
5. Document one transformation: input, change, reason, check, and effect on interpretation.

## Completion Criteria

Work is complete when the notebook runs top to bottom, preserves the raw data, shows at least one verification check, and contains a clear transformation note. Consult the [data-quality checklist](../../resources/data-quality-checklist.md).
