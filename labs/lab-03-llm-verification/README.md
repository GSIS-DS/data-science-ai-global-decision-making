# Lab 03: LLM Verification

This lab uses prepared examples for auditing AI-assisted outputs without paid APIs.

Tasks focus on checking generated code/claims against underlying data and documenting material AI use.

- Estimated time: 75 minutes
- Prerequisite: pandas filtering and summaries
- Output: audit table, corrected output, and AI Use Statement

## Prepared Audit

The starter includes a fictional AI-generated claim and code fragment. No external model or paid API is needed. Students must run independent checks against the synthetic dataset, identify the errors, revise the code or prose, and record what they accepted or rejected.

## Completion Criteria

- Every substantive claim is tied to a visible calculation or source row.
- Code is run rather than accepted by inspection alone.
- The corrected conclusion stays within the evidence.
- [AI_USE_TEMPLATE.md](AI_USE_TEMPLATE.md) is completed for the prepared exercise.
