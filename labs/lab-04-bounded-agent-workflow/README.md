# Lab 04: Bounded-Agent Workflow

This lab teaches a structured bounded workflow without complex autonomous agents or paid APIs.

Workflow:
Question -> Defined data/tools -> Analytical steps -> Draft output -> Verification -> Human decision

- Estimated time: 90 minutes
- Prerequisite: group summaries and responsible-AI policy
- Output: executed local workflow, verification record, and AI Workflow Card

## Tasks

1. Write a focused question and define the permitted dataset and functions.
2. Run a local function-based workflow that produces a draft regional comparison.
3. Apply explicit checks for row scope, missing values, and recalculated results.
4. Decide whether to accept, revise, or reject the draft and explain why.
5. Complete [AI_WORKFLOW_CARD_TEMPLATE.md](AI_WORKFLOW_CARD_TEMPLATE.md).

The notebook does not autonomously browse, call an API, or make a final decision. The student remains responsible for every analytical choice.
