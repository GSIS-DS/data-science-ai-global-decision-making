Use this short lab to confirm that Colab, pandas, the course dataset, and notebook saving work before assessed analysis begins.

- Estimated time: 30-40 minutes
- Prerequisite: GitHub and Google accounts
- Output: a saved notebook containing a data preview, one summary statistic, and one test chart

1. Open the starter in Colab and save a copy to Drive.
