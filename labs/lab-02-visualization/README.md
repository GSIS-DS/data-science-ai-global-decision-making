# Lab 02: Visualization (Week 5 Alignment)

This lab supports Week 5 exploratory analysis and visualization practice.

- Estimated time: 75 minutes
- Prerequisite: filtering and group summaries
- Output: one critique, one revised chart, and a 3-5 sentence interpretation

## Tasks

1. Reproduce the supplied misleading chart and identify the design problem.
2. Select a chart type that fits a cross-country or regional comparison.
3. Add title, axes, units, time period, and a synthetic-data source note.
4. Use adequate contrast and avoid relying on color alone.
5. Explain what the chart shows, what it does not show, and one relevant limitation without claiming causality.

## Completion Criteria

Use the [visualization checklist](../../resources/visualization-checklist.md). The revised figure must be accurate and accessible, and the prose must distinguish observation from explanation.
