# Global Indicators Sample Dataset Data Dictionary

This is a synthetic teaching dataset created for classroom exercises only. It should not be used for real empirical analysis or policy decisions.

| Field | Type | Description |
| --- | --- | --- |
| country | string | Country name |
| iso3 | string | ISO 3-letter code |
| year | integer | Observation year |
| gdp_growth | float | Annual GDP growth (%) |
| inflation | float | Inflation rate (%) |
| exports_usd_bn | float | Exports in USD billions |
| imports_usd_bn | float | Imports in USD billions |
| trade_openess_percent | float | Trade openness as percent of GDP |
| unemployment_percent | float | Unemployment rate (%) |
| exchange_rate_change_percent | float | Annual exchange rate change (%) |
| internet_users_percent | float | Internet users as percent of population |
| co2_per_capita | float | CO2 tons per capita |
| region | string | Region grouping |
