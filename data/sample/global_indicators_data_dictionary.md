# Global Indicators Sample Dataset - Data Dictionary

**This is a synthetic teaching dataset created for classroom exercises only.**

| Field | Type | Description |
|-------|------|-------------|
| country | string | Country name |
| iso3 | string | ISO 3-letter country code |
| year | integer | Year of observation (2020-2022) |
| gdp_growth | float | Annual GDP growth rate (percent) |
| inflation | float | Annual inflation rate (percent) |
| exports_usd_bn | float | Exports value in billions USD |
| imports_usd_bn | float | Imports value in billions USD |
| trade_openess_percent | float | Trade openness as % of GDP |
| unemployment_percent | float | Unemployment rate (percent) |
| exchange_rate_change_percent | float | Annual exchange rate change (percent) |
| internet_users_percent | float | Internet users as % of population |
| co2_per_capita | float | CO2 emissions per capita (metric tons) |
| region | string | Geographic region (Asia, Europe, North America, South America, Africa, Oceania)
