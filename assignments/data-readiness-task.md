# Data Readiness Tasks (DRT)

These tasks prepare students for applied analysis in this course. They are readiness and engagement activities, not a separate programming course.

## DRT 1: Python and Notebooks

- Timing: Before / during Week 2
- Check: Load a CSV; inspect rows and columns; calculate one summary statistic.

## DRT 2: Tabular Data

- Timing: Before Week 3
- Check: Create a subset and explain a data-quality issue.

## DRT 3: Statistics in Python

- Timing: Before Week 5
- Check: Produce one table and one chart with a short interpretation.

## DRT 4: GitHub Workflow

- Timing: During or immediately after Week 6
- Check: Update a notebook and submit a repository link.

Submission details are provided in weekly LMS and repository instructions.
