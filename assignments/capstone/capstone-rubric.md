# Capstone Rubric

| Category | Focus |
| --- | --- |
| Question quality and decision relevance | Clear, answerable, decision-relevant framing |
| Data quality and documentation | Source quality, cleaning logic, transparent metadata |
| Analytical appropriateness | Methods match question and data constraints |
| Reproducibility | Runnable workflow and documented steps |
| Interpretation and limitations | Appropriate claims and clear caveats |
| Visualization and communication | Readability, accessibility, decision relevance |
| Responsible AI use and verification | Material AI use disclosed and verified |
| Teamwork and professional presentation | Coordination, clarity, and professional standards |
| Individual contribution and understanding | Individual defense/reflection quality |
